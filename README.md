\## 🛣️ Overview



VINPAVE is a professional pavement design software that integrates with IITPAVE for flexible pavement analysis. It provides comprehensive tools for pavement design, strain analysis, and quantity calculation.



\### Features

\- \*\*6 Design Options\*\* for flexible pavements (Granular Base, CTB, RAP, etc.)

\- \*\*IITPAVE Integration\*\* for strain analysis

\- \*\*Theoretical Strain Calculations\*\* (Epz, Ept, Etcb)

\- \*\*Quantity \& Cost Estimation\*\* with currency conversion

\- \*\*Professional PDF Report Generation\*\*

\- \*\*Multi-currency Support\*\* (USD, INR, EUR, GBP, CAD, AUD)
